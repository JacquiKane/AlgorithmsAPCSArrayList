Determine The Average Value In An ArrayList

For an ArrayList of numbers, divide the sum of the numbers in the ArrayList, by the number of numbers in the ArrayList.

Let's use an example. Below, the ArrayList is called nums. The variable storing the average would be a double, as it may need to store a decimal value.


Sample Code - ArrayList Of Integers

                ArrayList<Integer> nums = new ArrayList<Integer>();
                // Add elements to the ArrayList
                // ....
                int total = 0;
                for (int num : nums)
                    total += num;
                        
                double avg = total / nums.size();

How About Finding The Average Of Object Attributes?

The same algorithm applies to ArrayList instances containing other objects. If you have an array of SoccerPlayer objects and you need the average age, then you would access the SoccerPlayer object's age attribute, add to a total, and then divide by the number of SoccerPlayer objects in the ArrayList.
