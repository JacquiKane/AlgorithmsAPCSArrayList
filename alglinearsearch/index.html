How Efficient Is Linear Search?

We say of the order of, designated by O(...), to describe the worst case in terms of comparisons. This is termed big O representation. The best case is described using big Omega notation, Ω(...).

The item might be found right away, if it is the first element in the ArrayList instance. In this case, the best case scenario is described as big Omega of 1, Ω(1). In the worst case, we have to compare each element in the ArrayList, n comparisons. This is designated using big O notation, as O(n).
