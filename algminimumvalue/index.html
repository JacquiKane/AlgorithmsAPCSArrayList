Determine Minimum Value In ArrayList

Looking for the minimum value in an ArrayList can be done sequentially, by comparing values in the ArrayList. Assuming the ArrayList instance is not sorted, we can start with the element at index 0, and take that to be the assumed minimum. We traverse the ArrayList sequentially, checking for values that are less than the minimum. Of course, if the ArrayList instance is already sorted and we know it is, then the minimum value will either be at the start in element 0, if the ArrayList is sorted in ascending order, or the last element, if the ArrayList is sorted in descending order.

Let's use an example, with the first element assumed to be the minimum value, until we find another value as we traverse an ArrayList of Integer objects, nums.


Sample Code, Assuming ArrayList Of Ints - nums

        ArrayList<Integer> nums = new ArrayList<Integer>();
        // Add elements to the ArrayList
        // ....                        
        int minVal = nums.get(0);
        for (int index = 1; index < nums.size(); index++) {
            if (minVal > nums.get(index) {
                minVal = nums.get(index);
                }
            }
